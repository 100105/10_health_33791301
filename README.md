# Sara's Health Clinic

This project is labwork for Dynamic web applications module. It is a health clinic booking system built using node.js with express, .ejs, and a mysql database.

---

## How to Run
1. npm install
2. node index.js

how to run on vm
1. npm install
2. npm install bcrypt
3. import database files: create_db.sql and insert_test_data.sql
3. git pull
4. forever stopall
5. forever start index.js
